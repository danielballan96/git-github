## Estados do Git

     [✅] Modificado (modified)
     [✅] Preparado (staged)
     [✅] Consolidado (committed)
    

## Comandos
     [✅] Git init
     [✅] git status
     [✅] git add <arquivo>
     [✅] git commit -m "Comentario"
     [✅] git log 
     [✅] git remote add origin <repo> 
     [✅] git remote -v
     [✅] git push -u origin  master
